# 教资高中数学学习记录网站 - 规格文档

## 1. 项目概述

- **项目名称**: 教资高中数学学习记录
- **项目类型**: 纯静态网页应用
- **核心功能**: 完整的高中数学教资备考记录工具，支持章节学习、笔记、错题、复习计划管理
- **目标用户**: 备考教师资格证高中数学的考生

## 2. 技术方案

- **技术栈**: 纯 HTML + CSS + JavaScript
- **数据存储**: 本地 JSON 文件 + localStorage 备份
- **运行方式**: 需要通过本地服务器（如 Live Server）运行，以支持 fetch 读取 JSON

## 3. 数据结构

```json
{
  "chapters": [
    {
      "id": "1-1",
      "name": "集合",
      "status": "completed", // pending, studying, completed
      "studyDate": "2026-04-01",
      "notes": []
    }
```

## 4. 界面设计

### 4.1 整体布局
- 左侧固定导航栏 (220px)
- 右侧主内容区域
- 响应式设计，适配移动端

### 4.2 配色方案 (清新舒适)
- 主色调: `#5B8DEF` (柔和蓝)
- 背景色: `#F5F7FA` (浅灰白)
- 卡片背景: `#FFFFFF`
- 文字主色: `#2C3E50`
- 文字次色: `#7F8C8D`
- 成功色: `#27AE60`
- 警告色: `#F39C12`
- 错误色: `#E74C3C`

### 4.3 字体
- 标题: "Noto Sans SC", sans-serif
- 正文: "Noto Sans SC", sans-serif

### 4.4 页面结构

1. **首页/概览**
   - 学习进度统计卡片
   - 最近学习章节
   - 待复习计划
   - 错题本提醒

2. **章节学习**
   - 高中数学知识树展示
   - 章节状态标记 (未学/学习中/已学完)
   - 点击查看章节详情

3. **学习笔记**
   - 笔记列表（支持搜索、筛选）
   - 笔记详情弹窗
   - 新增/编辑笔记

4. **错题本**
   - 错题列表（按章节分类）
   - 标记已掌握
   - 新增错题

5. **复习计划**
   - 计划日历视图
   - 计划列表
   - 新增/完成计划

## 5. 功能详细

### 5.1 数据读写
- 初始化时从 JSON 文件读取数据
- 数据变更时同时更新 JSON 文件（通过 fetch POST 到本地服务）和 localStorage
- 由于纯静态无法直接写文件，方案调整：
  - 使用 localStorage 作为主要存储
  - 提供导入/导出 JSON 功能，用户可手动备份

### 5.2 章节管理
- 预设高中数学知识点框架
- 支持标记学习状态
- 记录学习日期

### 5.3 笔记功能
- 支持标题、内容、标签
- 支持按章节筛选
- 支持搜索

### 5.4 错题功能
- 支持选择题、填空题、解答题
- 记录正确答案、错误答案、解析
- 支持标记已掌握

### 5.5 复习计划
- 支持设置复习日期
- 支持标记完成
- 显示待复习任务
